==================================================================
INSTALLATION README
==================================================================

Welcome to GeoNetwork opensource.

To run GeoNetwork opensource, you need the Java Runtime Environment (JRE)
version 1.5.0. You can download it from this url:

http://java.sun.com/javase/downloads/index_jdk5.jsp

Please refer to the changes.txt file for a list of known issues.
Refer to the online documentation or mailing-lists if you need 
more help at http://geonetwork-opensource.org
 
==================================================================
WINDOWS USERS
==================================================================

Under Windows, you can use the provided Windows installer that
will assist you even if you don't have Java VM installed.

==================================================================
OTHER USERS
==================================================================

Once you have downloaded the JRE, simply click on the .jar icon
to launch the installer. If this does not work, open a console
and issue:

java -jar geonetwork-install.jar

==================================================================
